# The Exodus Timeline Mystery

**Priority:** Post-MVP (Phase 2)
**Phase:** Phase 2, Weeks 1-6 (Ongoing narrative thread)
**Complexity:** High
**AI Integration:** High

## Overview

The Exodus Timeline Mystery is the central narrative thread of Phase 2. As players explore space searching for the human Exodus fleet, they discover fragments of a larger mystery: Why did humanity really leave Earth? Where did they go? What happened to them? The truth is discovered piece by piece through exploration, creating a compelling reason to keep playing and exploring.

## Core Concept

**The Mystery:**
Players start knowing only the official story: "Humanity left Earth in a massive exodus due to [catastrophic event]. They're out there somewhere."

**The Reality:**
Through discovery, players learn:
1. The official story has gaps and inconsistencies
2. There were multiple Exodus waves, not one
3. Some ships never made it
4. Others found something... unexpected
5. A few turned back with warnings
6. The truth is more complex and darker than believed

**Discovery Mechanics:**
- Find data logs in abandoned facilities
- Recover black boxes from derelict ships
- Interview alien species who encountered Exodus ships
- Piece together conflicting accounts
- Make theories, test them
- Uncover the truth gradually

## Design Principles

1. **Gradual Revelation** - Mysteries unfold slowly across many hours
2. **Player Agency** - Players piece together clues themselves
3. **Multiple Theories** - Evidence supports different interpretations
4. **Meaningful Discoveries** - Each clue feels significant
5. **Emotional Impact** - Truth has weight and consequence
6. **No Hand-Holding** - Players must think and connect dots
7. **Replayability** - Some clues are optional/hidden

## The Actual Mystery (Spoilers - For Design Only)

**What Really Happened:**

**2165:** Earth didn't face one catastrophe, but discovery of something that would change humanity forever. [Choose one based on desired tone]:

**Option A (Hope): The Signal**
- Earth detected alien signal containing coordinates to a "haven world"
- Signal included proof: images, data, instructions
- World appeared perfect for humanity
- Leadership decided to evacuate and start fresh
- Those who stayed believed it was a trap

**Option B (Fear): The Arrival**
- Something alien was coming to Earth
- Not immediately hostile, but fundamentally incompatible
- Exodus was escape from inevitable contact
- Those who stayed hoped to negotiate
- Some Exodus ships encountered "them" anyway

**Option C (Survival): The Decay**
- Earth was dying (climate/resource collapse)
- Terraforming attempt failed catastrophically
- Exodus was last hope for species
- Multiple destinations (hedging bets)
- Some destinations were viable, others weren't

**Option D (Division): The Schism**
- Humanity split on fundamental question
- Some wanted to merge with AI/become post-human
- Others wanted to stay "pure human"
- Exodus was voluntary separation
- Both groups believe they're saving humanity

**The Truth Has Layers:**
Each option has complexity:
- Multiple factions with different goals
- Conflicting intelligence about destinations
- Ships that reached destination vs ships that didn't
- Unexpected discoveries that changed plans
- Some made it, some failed, some transformed

1. **Days 50-70:** Something's not right (inconsistencies)

3. **Days 100-150:** Ships encountered something
4. **Days 150-200:** Different factions, different goals
5. **Days 200-250:** First contact with remnants
6. **Days 250+:** Full truth revealed, player chooses response

```python
from pydantic import BaseModel, Field
from typing import List, Dict, Optional
from datetime import datetime

class Evidence(BaseModel):
    evidence_id: str
    title: str
    type: str  # data_log, black_box, testimony, environmental, etc.
    content: str
    location_found: str
    day_found: int
    criticality: str  # minor, important, critical
    supports_theories: List[str] = []
    contradicts_theories: List[str] = []
    connections: List[str] = []  # Other evidence IDs this connects to
    revealed: bool = False  # Has player discovered this yet?

class Theory(BaseModel):
    theory_id: str
    name: str
    description: str
    supporting_evidence: List[str] = []  # Evidence IDs
    contradicting_evidence: List[str] = []
    plausibility: float = 0.5  # 0.0 to 1.0, based on evidence
    player_believes: bool = False

class ExodusWave(BaseModel):
    wave_id: str
    wave_number: int
    launch_date: str
    ship_count: int
    destination: str  # May be redacted or coordinates
    status: str  # departed, arrived, lost, unknown
    last_contact: Optional[str] = None
    last_message: Optional[str] = None
    fate: Optional[str] = None  # Discovered through play

class InvestigationState(BaseModel):
    evidence_collected: List[Evidence] = []
    theories: List[Theory] = []
    active_theory: Optional[str] = None
    exodus_waves: List[ExodusWave] = []
    revelations_unlocked: List[str] = []
    questions_raised: List[str] = []
    crew_theories: Dict[str, str] = {}  # crew_id -> theory_id

    def add_evidence(self, evidence: Evidence) -> List[str]:
        """Add evidence and return new questions raised"""
        self.evidence_collected.append(evidence)
        new_questions = []

        # Update theory plausibilities
        for theory in self.theories:
            if evidence.evidence_id in theory.supporting_evidence:
                theory.plausibility += 0.1
            if evidence.evidence_id in theory.contradicting_evidence:
                theory.plausibility -= 0.1
            theory.plausibility = max(0.0, min(1.0, theory.plausibility))

        # Check for revelations
        if evidence.criticality == "critical":
            new_questions = self._check_revelations(evidence)

        return new_questions
```

```gdscript
# investigation_manager.gd
class_name InvestigationManager
extends Node

signal evidence_discovered(evidence: Dictionary)
signal revelation_unlocked(revelation: String)

var theories: Array[Dictionary] = []

func add_evidence(evidence: Dictionary) -> void:
    """Add evidence to investigation"""
    evidence_collected.append(evidence)
    emit_signal("evidence_discovered", evidence)

    # Update theory plausibilities
    _update_theories_based_on_evidence(evidence)

    # Check for revelations
    if evidence.criticality == "critical":
        _check_for_revelations(evidence)

    # Save progress
    SaveManager.save_investigation()

func set_active_theory(theory_id: String) -> void:
    """Player chooses which theory they believe"""
    active_theory = theory_id

    # Update dialogues and crew discussions
    EventBus.emit_signal("player_theory_changed", theory_id)
```
