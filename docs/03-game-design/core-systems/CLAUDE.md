## Key Concepts
**XP Progression:** Exponential curve (100 XP for Level 2, 25,600 XP total for Level 10)
**Skills:** Engineering, Diplomacy, Combat, Science
**Ranks:** Cadet → Admiral
**Phase Structure:** Phase 1 (Earth) → Phase 2 (Space)
