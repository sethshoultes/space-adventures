# ship-systems - AI Agent Context

**Purpose:** 10 core ship systems and classification mechanics.

## Key Files
- ship-systems.md - All 10 systems, levels 0-5, power costs, mechanics
- ship-classification-system.md - 10 ship classes (Scout → Dreadnought)
- ship-documentation.md - Complete technical specifications

## When to Use
- Implementing ship systems
- Calculating power consumption
- Determining ship class
- Balancing system upgrades

## Key Concepts
**Power Budget:** Each system consumes power, Power Core generates it
**Levels:** 0 (broken) → 5 (maximum)
**Dependencies:** Some systems require others (e.g., shields need power)
**Classification:** Ship class based on system configuration
