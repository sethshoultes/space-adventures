2. Power Core - Energy generation
3. Propulsion - Sub-light engines

5. Life Support - Crew survival
6. Computer Core - AI assistance
7. Sensors - Detection/scanning
8. Shields - Defense
9. Weapons - Combat
10. Communications - Long-range comms
