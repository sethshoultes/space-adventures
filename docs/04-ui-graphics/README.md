## Overview

This directory contains comprehensive guidance for creating UI mockups and graphics using AI image generation (primarily DALL-E 3), as well as UI implementation specifications.

## Quick Start

**For AI Image Generation:**
1. Read [ui-graphics-getting-started.md](./ui-graphics-getting-started.md)
2. Use [ui-graphics-copy-paste-prompts.md](./ui-graphics-copy-paste-prompts.md) for quick results
3. Reference [ui-graphics-prompt-guide.md](./ui-graphics-prompt-guide.md) for customization

**For UI Implementation:**
1. Read [screen-designs.md](./screen-designs.md) for layouts
2. Check [visual-features.md](./visual-features.md) for specifications
3. Reference game design documents for context

## Design Style

**Star Trek TNG Inspired:**
- Clean, functional interfaces
- LCARS-style panels
- Muted colors (grays, blues, golds)
- Information density balanced with readability
- Retro-futuristic aesthetic
