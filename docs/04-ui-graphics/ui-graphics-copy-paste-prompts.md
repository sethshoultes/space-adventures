## CRITICAL TRANSPARENCY REQUIREMENTS

### For Ship Screens/Windows Viewing Space or Sky

**Rule:** Center viewing area must be **COMPLETELY TRANSPARENT** (PNG alpha = 0)
**Why:** So space backgrounds, sky, or mission scenes can show through

### For Button Placement in Ship

**Rule:** Background panels should be **SEMI-TRANSPARENT** (40-70% opacity)
**Why:** Creates holographic overlay effect, shows environment behind

### For Standalone Buttons/Menus

**Rule:** Elements themselves are **OPAQUE** (100% solid)
**Why:** Buttons need to be clearly visible and clickable

## POST-PROCESSING STEPS (If Needed)

### Fix Transparency in GIMP

```
1. Open generated PNG in GIMP
2. Right-click layer → Add Alpha Channel
3. Select → By Color → Click area that should be transparent
4. Edit → Clear (or press Delete key)
5. For semi-transparency: Layer → Transparency → Threshold Alpha
6. File → Export As → PNG
7. Options: Check "Save background color", Compression 9
8. Export
```

### Verify Transparency

```
- Open in image viewer with checkerboard background
- Transparent areas show checkerboard pattern
- Semi-transparent areas show partial checkerboard
- Opaque areas show no checkerboard
```
