## Port Registry (NCC-1701)

- **17010** - Gateway (NCC-1701-0 / USS Enterprise)
- **17011** - AI Service (NCC-1701-1)
- **17012** - Whisper (NCC-1701-2, optional)
- **17014** - Redis (NCC-1701-4)

**Rationale:** Avoid conflicts with common ports (8000-8100), memorable, thematic

## Settings System

Game configuration including:
- Graphics settings
- Audio settings
- Gameplay preferences
- Control bindings
- Accessibility options
