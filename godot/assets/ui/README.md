# UI Assets Directory

This directory contains all generated UI graphics for Space Adventures.

## Directory Structure

```
ui/
├── _catalog/           Asset inventory and tracking
├── backgrounds/        Dashboard backgrounds (1280x720)
├── buttons/            LCARS buttons (various sizes/colors)
├── frames/             Viewscreen and display frames
├── panels/             Information panels and containers
├── indicators/         Status bars, badges, alerts
└── overlays/           Modals, loading screens, tooltips
```

## Asset Requirements

- **Format:** PNG with alpha channel (RGBA)
- **Style:** Star Trek LCARS / retro sci-fi
- **Quality:** Professional, clean edges
- **Size:** Optimized (<500KB each)
- **Transparency:** As specified per asset type
